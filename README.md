# Kurup & Associates - Premium Legal Data Management System

Welcome to the Kurup & Associates project. This is a premium, desktop-first web application for legal office management.

## Getting Started

Please follow these steps to run the application:

1.  **Install Dependencies**:
    Open your terminal in this directory (`c:\Users\ASUS\OneDrive\Documents(1)\coding\project-Kurup&Co`) and run:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    Run the following command to start the Vite development server:
    ```bash
    npm run dev
    ```

3.  **Open in Browser**:
    Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

-   **Frontend**: React (Vite) + TypeScript
-   **Styling**: Tailwind CSS + Lucide React (Icons)
-   **Backend**: Supabase (Client configured)
-   **Routing**: React Router DOM

## Project Structure

-   `src/components/ui`: Reusable UI components (Button, Input, Card, etc.)
-   `src/components/layout`: Layout components (Sidebar, Header)
-   `src/pages`: Application pages (Login, Dashboard)
-   `src/context`: React Context providers (AuthContext)
-   `src/lib`: Utility functions and Supabase client
